Powerful jQuery Supplementary Menu Builder

== Description ==
<h4>
Enhance your site by adding an elegant supplementary menu.
</h4>
<p>
WYSIWYG - Easily build supplementary menus for your site using this great tool. Add links with custom icon and built in animations to open and close in the form.
</p>
<p> The main idea is to have a trigger that when you hover or click on it will display an auxiliary menu in the form of small circles. No coding experience is necessary, everything is done through
the UI, which is very intuitive and easy to use. Add your own icons for the links and also for the opening trigger.
</p>

<h4>Features</h4>
<ul>
<li>Create unlimited menus</li>
<li>Full range of properties</li>
<li>Insert anywhere on your site using shortcode</li>
<li>Built-in Effects</li>
<li>Fully configurable</li>
<li>WYSIWYG editor - Live preview</li>
<li>Changes propagate automatically</li>
</ul>

<p>For a live demo visit the plugin's main site <a href='http://wpbubblemenu.com' target='blank'>http://wpbubblemenu.com</a></p>


== Installation ==

1. Upload and extract the zip file downloaded to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
